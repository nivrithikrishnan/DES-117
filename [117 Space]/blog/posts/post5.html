Blog Post 5

November 21st, 2022

In this course, we covered the basics of HTML and CSS, and how they work together with the browser to display a webpage. The tech industry has grown exponentially over the past several years, causing us to become more dependent on everyday technology, and increased the need for browsers and designs to be both accessible and efficient. To keep up with this movement, new features are constantly being added and changed to web development programs.

Backwards compatibility allows all web pages to be resilient over time. Allowing CSS to be an optional, presentation layer increases accessibility, and allows the focus of a web page to be on content rather than design. A metaphor mentioned in the video compares CSS to stage directions, and HTML as a script or stage. HTML is contextual, wheras CSS is intentional, adaptable and built to work around the context. As the stage changes, so must the directions.

Web development, UI/UX, and any technology field really is constantly and rapidly changing.
